This project provides a deep learning framework for vehicle trajectory prediction, designed as a modular and extensible platform for time series modeling. The core contribution is the Attention Residuals (AttnRes) mechanism, which replaces traditional residual connections with an attention-based adaptive fusion strategy, enabling dynamic weighting of multi-level features.

The framework integrates several mainstream time series models (e.g., DLinear, FEDformer, PatchTST, TimeMixer) under a unified interface, allowing efficient model comparison and structural experimentation.
